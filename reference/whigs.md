# Membership network of American Whigs.

These data record the membership of 136 colonial Americans in 5 Whig
organizations.

## Format

An affiliation network; see
[is_an](http://corybrunson.github.io/bitriad/reference/affiliation_network.md).

## Source

Fischer, D.H. (1994). *Paul Revere's Ride*. Oxford University Press.
